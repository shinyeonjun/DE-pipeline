# 유튜브 분석 지표 용어 사전

## 기본 성과 지표

### 조회수 (Views)
동영상 재생 횟수입니다. 유튜브는 30초 이상 시청을 유효 조회로 인정합니다.
- **고유 조회수**: 중복 제거된 순수 시청자 수
- **반복 조회**: 같은 사용자의 재시청 (일정 시간 간격 후 인정)

### 시청 시간 (Watch Time)
총 시청된 시간(분)입니다. 유튜브 알고리즘에서 **가장 중요한 단일 지표**입니다.
- **누적 시청 시간**: 채널 전체 성과 지표
- **평균 시청 시간**: 영상당 평균 시청 분

### 구독자 수 (Subscribers)
채널을 구독한 사용자 수입니다.
- **순증가**: 신규 구독 - 구독 취소
- **Growth Rate**: (현재 - 이전) / 이전 × 100

## 참여 지표 (Engagement Metrics)

### 참여율 (Engagement Rate)
시청자가 콘텐츠에 반응한 정도를 측정합니다.

**표준 공식:**
```
참여율(%) = (좋아요 + 댓글) / 조회수 × 100
```

**확장 공식 (공유 포함):**
```
참여율(%) = (좋아요 + 댓글 + 공유) / 조회수 × 100
```

**벤치마크:**
| 수준 | 참여율 |
|:---|:---|
| 매우 우수 | 5% 이상 |
| 우수 | 2-5% |
| 평균 | 1-2% |
| 개선 필요 | 1% 미만 |

### 클릭률 (CTR: Click-Through Rate)
노출 대비 클릭 비율입니다. 썸네일과 제목의 효과를 측정합니다.

**공식:**
```
CTR(%) = 클릭 수 / 노출 수 × 100
```

**벤치마크:**
| 수준 | CTR |
|:---|:---|
| 매우 우수 | 10% 이상 |
| 우수 | 5-10% |
| 평균 | 2-5% |
| 개선 필요 | 2% 미만 |

### 시청 지속률 (Retention Rate)
영상의 어느 지점까지 시청하는지를 백분율로 표시합니다.

**주요 구간:**
- **30초 지점**: 첫인상 평가 (80% 이상 권장)
- **50% 지점**: 콘텐츠 품질 지표 (60% 이상 권장)
- **종료 지점**: 완주율 (40% 이상 권장)

## 성장 지표

### View Velocity (조회수 속도)
단위 시간당 조회수 증가량입니다. 트렌딩 알고리즘의 핵심 지표입니다.

**공식:**
```
View Velocity = Δ조회수 / Δ시간(시간)
```

**트렌딩 기준 (추정):**
- **상위권**: 10만+/시간
- **진입 가능**: 1만+/시간
- **일반**: 1천 미만/시간

### 성장률 (Growth Rate)
특정 기간 동안의 변화율입니다.

**공식:**
```
성장률(%) = (현재값 - 이전값) / 이전값 × 100
```

### RPM (Revenue Per Mille)
1,000회 조회당 수익입니다. 크리에이터가 실제로 받는 금액 기준입니다.

**카테고리별 평균 RPM (한국):**
| 카테고리 | RPM 범위 |
|:---|:---|
| 금융/투자 | $3-10 |
| 교육 | $2-6 |
| 게임 | $1-3 |
| 엔터테인먼트 | $1-3 |
| 쇼츠 | $0.01-0.05 |

## 트렌딩 관련 지표

### 트렌딩 순위 (Trending Rank)
유튜브 트렌딩 탭에서의 위치입니다. 1위가 최상위입니다.
- **순위 범위**: 1-50위 (국가별 상이)
- **갱신 주기**: 약 15분

### 순위 변동 (Rank Movement)
이전 스냅샷 대비 순위 변화입니다.
- **상승 (↑)**: 순위 상승
- **하락 (↓)**: 순위 하락
- **유지 (-)**: 변동 없음
- **신규 (NEW)**: 새로 진입

### 트렌딩 체류 시간 (Trending Duration)
트렌딩 목록에 머무는 총 시간입니다.
- **단기**: 1-6시간
- **중기**: 6-24시간
- **장기**: 24시간 이상 (바이럴 콘텐츠)

## 콘텐츠 분류

### 쇼츠 (Shorts)
60초 이하, 세로 형식(9:16)의 짧은 동영상입니다.
- **최대 길이**: 60초
- **비율**: 9:16 (세로형)
- **특징**: 반복 재생, 스와이프 탐색

### 일반 영상 (Long-form Video)
60초를 초과하는 표준 동영상입니다.
- **권장 길이**: 8-15분 (수익화 최적)
- **비율**: 16:9 (가로형)
- **특징**: 광고 삽입, 챕터 마커

### 라이브 스트림 (Live Stream)
실시간 방송입니다.
- **특징**: 실시간 채팅, 슈퍼챗
- **노출**: 구독자 알림, 홈 피드 상단

## 채널 지표

### 채널 권위도 (Channel Authority)
채널의 신뢰성과 영향력을 나타내는 복합 지표입니다.
- **구독자 수**: 규모 지표
- **평균 조회수**: 콘텐츠 성과
- **업로드 일관성**: 활동성
- **과거 트렌딩 경험**: 알고리즘 신뢰도

### 구독자당 조회수 (Views per Subscriber)
채널의 콘텐츠 효율성을 측정합니다.

**공식:**
```
구독자당 조회수 = 평균 조회수 / 구독자 수
```

**벤치마크:**
- **우수**: 30% 이상
- **평균**: 10-30%
- **개선 필요**: 10% 미만
