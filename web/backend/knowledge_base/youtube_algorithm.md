# 유튜브 추천 알고리즘 심층 분석

## 개요

유튜브의 추천 시스템은 크게 **트렌딩(Trending)**, **홈 피드(Home Feed)**, **추천 동영상(Suggested Videos)** 세 가지로 나뉩니다. 각각 다른 알고리즘 가중치를 사용하며, 크리에이터와 마케터가 이를 이해하면 전략적으로 콘텐츠 노출을 극대화할 수 있습니다.

---

## 1. 트렌딩 알고리즘

### 핵심 메커니즘

트렌딩은 **현재 진행형 관심도**를 측정합니다. 단순 조회수가 아닌 **시간당 성장률(View Velocity)**이 핵심입니다.

**가중치 공식 (추정):**
```
트렌딩 점수 = (조회수 성장률 × 0.4) + (참여율 × 0.3) + (채널 신뢰도 × 0.2) + (지역 관련성 × 0.1)
```

### 주요 지표 상세

| 지표 | 정의 | 기준값 | 영향도 |
|:---|:---|:---|:---|
| **View Velocity** | 시간당 조회수 증가량 | 10만+/시간 (상위권) | 40% |
| **Engagement Rate** | (좋아요+댓글)/조회수 | 2% 이상 우수 | 30% |
| **Watch Time** | 평균 시청 지속률 | 50% 이상 권장 | 20% |
| **Channel Authority** | 과거 성과 + 구독자 | 트렌딩 경험 유무 | 10% |

### 트렌딩 진입 조건

1. **초기 부스트**: 업로드 후 1-2시간 내 급격한 조회수 상승
2. **광범위성**: 특정 니치가 아닌 대중적 관심 주제
3. **안전성**: 커뮤니티 가이드라인 위반 없음
4. **다양성**: 같은 채널의 연속 트렌딩 제한

---

## 2. 홈 피드 추천 알고리즘

### 개인화 메커니즘

홈 피드는 **협업 필터링(Collaborative Filtering)**과 **콘텐츠 기반 필터링(Content-Based Filtering)**을 결합합니다.

**주요 신호:**
- 시청 기록 및 검색 기록
- 좋아요/싫어요 반응
- 채널 구독 여부
- 시청 완료율
- 유사 사용자 행동 패턴

### 최적화 전략

1. **첫 30초**: 시청자 이탈 방지가 핵심. 후킹(Hooking) 기법 필수
2. **세션 시간**: 연관 동영상 유도로 세션 길이 증가
3. **CTR 최적화**: 썸네일/제목 A/B 테스트 권장

## 3. 쇼츠 vs 일반 영상 알고리즘 차이

### 플랫폼 분리

유튜브는 쇼츠와 일반 영상을 **별도 추천 시스템**으로 운영합니다.

| 특성 | 쇼츠 (Shorts) | 일반 영상 (Long-form) |
|:---|:---|:---|
| **노출 위치** | 쇼츠 피드, 홈 하단 | 홈 피드, 검색, 추천 |
| **알고리즘 가중치** | 완주율 > 좋아요 | 시청시간 > CTR |
| **수익화** | RPM 낮음 ($0.01-0.05) | RPM 높음 ($2-10) |
| **구독 전환** | 낮음 (3-5%) | 높음 (10-20%) |
| **트렌딩 진입** | 제한적 | 가능 |

### 전략적 활용

- **쇼츠**: 신규 시청자 유입, 채널 인지도 확보
- **일반 영상**: 수익화, 구독자 전환, 커뮤니티 형성
- **하이브리드**: 쇼츠로 유입 → 일반 영상으로 전환 유도

## 4. 알고리즘 최적화 체크리스트

### 업로드 전
- [ ] 키워드 리서치 (Google Trends, VidIQ)
- [ ] 경쟁 영상 분석 (상위 10개 벤치마크)
- [ ] 썸네일 3종 이상 준비
- [ ] 제목 SEO 최적화 (핵심 키워드 앞배치)

### 업로드 후 (골든 타임 48시간)
- [ ] 커뮤니티 탭 알림 발송
- [ ] SNS 크로스 포스팅
- [ ] 첫 50개 댓글에 직접 답변
- [ ] 성과 모니터링 (1시간/6시간/24시간)

### 지속적 관리
- [ ] 주 1회 이상 일관된 업로드
- [ ] 구독자 행동 패턴 분석
- [ ] 저성과 콘텐츠 원인 분석
- [ ] 트렌드 변화 모니터링

## 5. 자주 묻는 질문

**Q: 조회수가 높으면 무조건 트렌딩에 오르나요?**
A: 아닙니다. 조회수보다 **성장 속도**가 중요합니다. 1주일에 1000만보다 1시간에 100만이 더 유리합니다.

**Q: 쇼츠가 일반 영상 알고리즘에 영향을 주나요?**
A: 직접적 영향은 제한적입니다. 유튜브는 두 시스템을 분리 운영합니다. 다만 채널 전체 성과는 간접 영향을 줄 수 있습니다.

**Q: 업로드 시간이 트렌딩에 영향을 주나요?**
A: 네. 타겟 시청자의 **활동 피크 시간 2-3시간 전** 업로드가 권장됩니다. 한국 기준 오후 4-5시 업로드 → 저녁 7-10시 노출 극대화.
